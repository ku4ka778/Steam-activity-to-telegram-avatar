# Steam-activity-to-telegram-avatar
This app allows you to broadcast your Steam activity to your Telegram avatar.  The original avatar remains. Only text is added over it. (Currently offline in Steam/Currently online in Steam/Playing "game name" in Steam)

You should fill the 'setting.ini' manually. 
This program has logs in 'logs.txt' so u can check the main information.

The avatar is updated every 5 minutes.
